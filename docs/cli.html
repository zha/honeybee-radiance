Honeybee Radiance Command Line Interface¶

honeybee_radiance¶

honeybee radiance commands.

honeybee_radiance [OPTIONS] COMMAND [ARGS]...

grid¶

Commands for generating and modifying sensor grids.

honeybee_radiance grid [OPTIONS] COMMAND [ARGS]...

merge¶

Merge several radiance grid files into a single file.

Args:
input_folder: Input folder.
base_name: File base name. All of the files must start with base name and
continue with _ and an integer values.
extension: File extention. [Default: .pts]
honeybee_radiance grid merge [OPTIONS] INPUT_FOLDER BASE_NAME [EXTENSION]

Options

--folder <folder>¶

Optional output folder. [default: .]

Arguments

INPUT_FOLDER¶

Required argument

BASE_NAME¶

Required argument

EXTENSION¶

Optional argument

split¶

Split a radiance grid file into smaller grids based on maximum sensor count.

Args:
grid-file: Full path to input sensor grid file.
count: Maximum number of sensors in new files. The number will be rounded to
closest round number for each file. For example if the input file has 21
sensors and input count is set to 5 this command will generate 4 files where
the first three files will have 5 sensors and the last file will have 6.
honeybee_radiance grid split [OPTIONS] GRID_FILE COUNT

Options

--folder <folder>¶

Optional output folder. [default: .]

Arguments

GRID_FILE¶

Required argument

COUNT¶

Required argument

sky¶

Commands to generate Radiance skies.

honeybee_radiance sky [OPTIONS] COMMAND [ARGS]...

illuminance¶

Generate an overcast / cloudy sky with certain illuminance value.

illum: Desired illuminance value in lux. [default: 100000].

honeybee_radiance sky illuminance [OPTIONS] [ILLUM]

Options

--folder <folder>¶

Output folder. [default: .]

--name <name>¶

Sky file name.

Arguments

ILLUM¶

Optional argument
